# Santi POS web

Un cliente hibrido POS no tan cualquiera, muy a medida.

Diseño para **minimizar tiempos de ventas**, para uso en dispositivos móviles, tecnología de almacenamiento híbrido (a futuro | desarrollo no público), implementado para ser usado solo con teclado númerico independiente e impresora de 58mm, como mínimo.

- Ideal para carnicerías no digitalizadas y con máquinas de cobro digital
- Sin conexión a SII
  
[visitar aquí](https://fabocv.github.io/santi-pos)
